# Claude 입문 가이드: 똑똑하게 대화하고 협업하기

Anthropic에서 개발한 Claude는 뛰어난 추론 능력과 인간 친화적인 대화 스타일로 사랑받는 AI 모델입니다. 이 가이드를 통해 Claude의 강점을 100% 활용하는 방법을 알아보세요.

## 1. Claude의 3가지 모델 이해하기

Claude는 사용자의 용도와 작업의 복잡도에 따라 선택할 수 있는 세 가지 모델을 제공합니다. 

가장 가볍고 빠른 모델인 Haiku(하이쿠)는 단순한 문서 요약이나 빠른 답변이 필요할 때 적합합니다. 반면 Sonnet(소네트)은 지능과 속도의 균형이 가장 잘 맞춰진 모델로, 코딩, 글쓰기, 복잡한 데이터 분석 등 대부분의 일상적인 작업에 폭넓게 추천됩니다. 마지막으로 Opus(오푸스)는 가장 강력한 지능을 갖춘 모델로, 고도의 전략 수립이나 깊은 철학적 사고가 요구되는 복잡한 문제 해결에 사용됩니다.

## 2. 'Artifacts' 기능 활용하기

Claude의 가장 큰 차별점 중 하나는 대화창 옆에 별도로 띄워지는 Artifacts 창입니다. 코드를 작성하거나, 웹사이트 디자인을 하거나, 긴 문서를 작성할 때 결과물을 대화와 분리하여 독립적인 화면으로 보여줍니다.

이를 통해 사용자는 작성된 코드가 실제 웹 환경에서 어떻게 작동하는지 실시간으로 렌더링하여 확인할 수 있습니다. 또한, 결과물을 보면서 "이 부분의 색깔을 파란색으로 바꿔줘"와 같이 대화로 수정 사항을 요청하면, Artifacts 창의 결과물이 즉각적으로 업데이트되어 매우 직관적인 협업이 가능합니다.

## 3. 프로젝트(Projects) 기능으로 나만의 AI 만들기

Pro 또는 Team 플랜 사용자는 프로젝트(Projects) 기능을 통해 Claude를 특정 업무에 최적화된 맞춤형 AI로 만들 수 있습니다. 자주 사용하는 참고 자료, 예를 들어 회사 가이드라인 PDF나 핵심 코드 파일들을 프로젝트 공간에 미리 업로드해 둘 수 있습니다.

이렇게 설정된 프로젝트 내에서 대화를 시작하면, Claude는 업로드된 모든 자료의 맥락을 완벽히 숙지한 상태로 답변을 생성합니다. 예를 들어 회사의 마케팅 브랜드 가이드를 올려두고 신제품 카피를 요청하면, 회사의 톤앤매너에 정확히 부합하는 결과물을 얻을 수 있습니다.

## 4. 좋은 답변을 얻는 꿀팁

Claude에게서 원하는 품질의 답변을 이끌어내기 위해서는 역할 부여와 예시 제공이 매우 중요합니다. Claude는 이러한 맥락 정보에 매우 민감하게 반응하여 답변의 톤과 구조를 조절합니다.

대화를 시작할 때 "너는 10년 차 전문 카피라이터야"와 같이 구체적인 역할을 정해주면 답변의 전문성이 높아집니다. 또한, "다음과 같은 스타일로 써줘"라며 원하는 결과물의 예시 문장이나 구조를 함께 제공하면, 의도한 바에 정확히 부합하는 결과물을 얻을 확률이 비약적으로 상승합니다.
