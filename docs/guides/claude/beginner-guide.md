# Claude 입문 가이드: 똑똑하게 대화하고 협업하기

Anthropic에서 개발한 Claude는 뛰어난 추론 능력과 인간 친화적인 대화 스타일로 사랑받는 AI 모델입니다. 이 가이드를 통해 Claude의 강점을 100% 활용하는 방법을 알아보세요.

## 1. Claude의 3가지 모델 이해하기

Claude는 용도에 따라 세 가지 모델을 선택할 수 있습니다.

- **Haiku (하이쿠)**: 가볍고 빠릅니다. 단순한 요약이나 빠른 답변이 필요할 때 적합합니다.
- **Sonnet (소네트)**: 지능과 속도의 균형이 완벽합니다. 코딩, 글쓰기, 복잡한 분석 등 대부분의 작업에 추천합니다.
- **Opus (오푸스)**: 가장 강력한 지능을 가졌습니다. 고도의 전략 수립이나 깊은 철학적 사고가 필요한 경우에 사용합니다.

## 2. 'Artifacts' 기능 활용하기

Claude의 가장 큰 차별점은 **Artifacts** 창입니다. 코드를 작성하거나, 웹사이트 디자인을 하거나, 긴 문서를 쓸 때 대화창 옆에 별도의 화면으로 결과물을 보여줍니다.
- **실시간 확인**: 작성된 코드가 어떻게 작동하는지 바로 볼 수 있습니다.
- **수정 요청**: "이 부분 색깔을 파란색으로 바꿔줘"라고 하면 Artifacts 창에서 즉시 반영됩니다.

## 3. 프로젝트(Projects) 기능으로 나만의 AI 만들기 (Pro/Team 플랜)

자주 사용하는 참고 자료(PDF, 코드 파일, 가이드라인)를 '프로젝트'에 업로드해두면, Claude가 해당 맥락을 완벽히 이해한 상태에서 대화를 시작할 수 있습니다.
- **사례**: 우리 회사 마케팅 가이드라인을 올려두고 "이번 신제품 카피 써줘"라고 요청하기.

## 4. 좋은 답변을 얻는 꿀팁

Claude는 **'역할 부여'**와 **'예시 제공'**에 매우 민감하게 반응합니다.
- "너는 10년 차 전문 카피라이터야."라고 역할을 정해주세요.
- "다음과 같은 스타일로 써줘: [예시 문장]" 처럼 예시를 주면 품질이 비약적으로 상승합니다.
