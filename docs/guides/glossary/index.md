# AI 필수 용어 사전 (Glossary)

AI 서비스를 처음 접할 때 가장 큰 장벽은 생소한 전문 용어들입니다. 이 페이지에서는 AI 관련 문서나 가이드를 읽을 때 자주 등장하는 핵심 용어들을 초보자의 눈높이에 맞추어 쉽게 설명합니다.

## 1. 모델과 작동 원리 관련 용어

AI가 어떻게 생각하고 작동하는지를 이해하는 데 필요한 기본적인 용어들입니다.

| 용어 | 영문 | 설명 |
| --- | --- | --- |
| **대규모 언어 모델** | LLM (Large Language Model) | 수많은 텍스트 데이터를 학습하여 인간처럼 문장을 이해하고 생성할 수 있는 인공지능 모델입니다. ChatGPT, Claude, Gemini가 모두 여기에 속합니다. |
| **매개변수** | Parameter | AI 모델의 뇌세포와 같은 역할을 하는 수치입니다. 매개변수가 많을수록 모델의 지능이 높고 복잡한 문제를 잘 풀 수 있지만, 실행하는 데 더 많은 컴퓨터 자원이 필요합니다. |
| **토큰** | Token | AI가 글자를 읽고 쓰는 기본 단위입니다. 한글의 경우 1글자가 1~2토큰, 영어는 1단어가 보통 1토큰으로 계산됩니다. AI 서비스의 요금은 보통 이 토큰 사용량을 기준으로 매겨집니다. |
| **컨텍스트 윈도우** | Context Window | AI가 한 번의 대화에서 기억하고 참고할 수 있는 텍스트의 최대 분량입니다. 컨텍스트 윈도우가 클수록 수백 페이지의 책이나 긴 코드를 한 번에 읽고 답변할 수 있습니다. |

## 2. 사용자 상호작용 관련 용어

우리가 AI에게 일을 시키고 결과를 받아보는 과정에서 쓰이는 용어들입니다.

| 용어 | 영문 | 설명 |
| --- | --- | --- |
| **프롬프트** | Prompt | AI에게 내리는 질문이나 지시문입니다. "이 문장을 번역해줘"라는 짧은 문장부터, 구체적인 조건이 달린 긴 글까지 모두 프롬프트에 해당합니다. |
| **환각 현상** | Hallucination | AI가 사실이 아닌 내용을 마치 진짜인 것처럼 그럴싸하게 꾸며내어 대답하는 현상입니다. AI의 답변을 항상 교차 검증해야 하는 이유입니다. |
| **에이전트** | Agent | 단순한 질문-답변을 넘어, 주어진 목표를 달성하기 위해 스스로 계획을 세우고 도구(브라우저, 코드 등)를 사용하여 작업을 완수하는 자율적인 AI 시스템입니다. Manus가 대표적인 예입니다. |

## 3. 고급 기능 및 기술 용어

AI 서비스를 더 똑똑하게 활용하기 위해 알아두면 좋은 심화 용어들입니다.

| 용어 | 영문 | 설명 |
| --- | --- | --- |
| **멀티모달** | Multimodal | 텍스트뿐만 아니라 이미지, 오디오, 비디오 등 다양한 형태의 데이터를 동시에 이해하고 처리할 수 있는 능력입니다. |
| **검색 증강 생성** | RAG (Retrieval-Augmented Generation) | AI가 답변을 생성하기 전에, 먼저 외부 데이터베이스나 인터넷을 검색하여 최신 정보나 회사 내부 문서를 참고하도록 만드는 기술입니다. 환각 현상을 줄이는 데 매우 효과적입니다. |
| **미세 조정** | Fine-tuning | 이미 똑똑하게 학습된 기본 AI 모델에 특정 분야의 데이터(예: 법률, 의료)를 추가로 학습시켜, 그 분야의 전문가로 만드는 과정입니다. |
| **온디바이스 AI** | On-device AI | 인터넷을 통해 클라우드 서버에 접속하지 않고, 스마트폰이나 노트북 등 기기 내부에서 직접 실행되는 AI입니다. 처리 속도가 빠르고 개인정보 보호에 유리합니다. |

이 용어 사전은 새로운 기술 트렌드가 등장함에 따라 지속적으로 업데이트될 예정입니다. 가이드를 읽다가 모르는 단어가 나오면 언제든 이 페이지를 참고해 주세요.
