# Manus 입문 가이드: 첫걸음부터 자동화까지

Manus는 단순한 챗봇을 넘어, 사용자의 목표를 달성하기 위해 스스로 계획을 세우고 도구를 사용하는 **자율형 AI 에이전트**입니다. 이 가이드는 Manus를 처음 접하는 분들이 핵심 기능을 이해하고 바로 활용할 수 있도록 돕습니다.

## 1. Manus란 무엇인가요?

기존의 AI가 질문에 답을 하는 '상담원'이었다면, Manus는 요청한 일을 끝까지 완수하는 '대리인'에 가깝습니다.

가장 큰 특징은 스스로 생각하고 계획을 수립한다는 점입니다. 예를 들어 "제주도 3박 4일 여행 계획 짜줘"라고 요청하면, Manus는 항공권 조회, 숙소 예약 가능 여부 확인, 맛집 리스트 정리 등 필요한 세부 단계를 스스로 설계합니다. 

또한, 계획을 실행하기 위해 도구를 직접 사용합니다. 사람처럼 웹 브라우저를 열어 최신 정보를 검색하고, 코드를 작성하여 데이터를 분석하며, 최종 결과물을 문서나 파일 형태로 생성하여 제공합니다.

## 2. 첫 번째 작업 요청하기 (프롬프트 팁)

Manus에게 일을 시킬 때는 **'결과물'**과 **'제약 사항'**을 명확히 알려주는 것이 좋습니다.

> **추천 예시:**
> "최근 1주일간의 엔비디아(NVDA) 주가 흐름을 분석해서 표로 정리해줘. 그리고 향후 전망에 대한 주요 뉴스 3가지를 요약해서 보고서 형식으로 작성해줘."

## 3. Manus의 핵심 기능 활용하기

| 기능 | 활용 사례 |
| --- | --- |
| **웹 브라우징** | 최신 뉴스 검색, 실시간 주식/코인 가격 확인, 상품 가격 비교 |
| **코드 실행** | 엑셀 파일 데이터 분석, 복잡한 수학 계산, 그래프 생성 |
| **파일 관리** | 보고서(PDF/Markdown) 작성, 이미지 생성, 데이터 시트(CSV) 저장 |

## 4. 진행 상황 모니터링

Manus는 작업을 수행하는 동안 '생각 중(Thinking)' 단계와 '실행 중(Executing)' 단계를 사용자에게 실시간으로 보여줍니다. 이를 통해 에이전트가 현재 어떤 의도로 어떤 도구를 사용하고 있는지 투명하게 파악할 수 있습니다. 

만약 모니터링 중 Manus가 사용자의 의도와 다르게 잘못된 방향으로 작업을 진행하고 있다면, 언제든지 중간에 개입하여 방향을 수정하거나 새로운 지침을 내릴 수 있습니다.
