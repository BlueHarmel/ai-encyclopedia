# Verification Checklist

## Fast Path: Official Sources

공식 문서, 공식 릴리즈 노트, 공식 가격 페이지, 공식 뉴스룸은 신뢰도 100%로 취급한다.

- URL과 접근 날짜를 기록한다.
- 변경 적용 범위를 서비스 페이지에 적는다.
- 예전 내용과 충돌하면 예전 내용을 `superseded`로 표시한다.
- 별도의 외부 검증 기사나 커뮤니티 확인은 필수가 아니다.

## Slow Path: Signals

X 포스트, 뉴스 기사, 커뮤니티 글, Reddit, Discord, 비공식 블로그와 유튜브는 검증 없이 사실로 승격하지 않는다.

## Promotion Rules

- 공식 출처에 같은 내용이 있으면 `fact`로 승격한다.
- 공식 출처가 없지만 반복적으로 관찰되면 `observed-signal`로 둔다.
- 사용자의 실제 환경에서 확인된 내용은 `local-observation`으로 둔다.
- 비밀, 토큰, 개인 채팅, 내부 로그 원문은 공개 문서에 싣지 않는다.
