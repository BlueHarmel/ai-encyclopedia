# GitHub Copilot / Plans And Billing

Last verified: 2026-04-26

## Summary

This page compares current GitHub Copilot plans from GitHub Docs. It includes individual, student, organization, and enterprise plans in one table while keeping seat-based billing and premium request allowances explicit.
