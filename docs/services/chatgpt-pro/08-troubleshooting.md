## 증상별 트러블슈팅 표

| 증상 | 가능 원인 | 해결 가이드 |
|---|---|---|
| 답변이 추상적임 | 출력 포맷 미고정 | 표/코드/체크리스트 강제 |
| 코드 제안이 위험함 | 범위/제약 미지정 | 수정 파일 화이트리스트 지정 |
| 테스트 누락 | 검증 단계 미지시 | 테스트/린트/빌드 필수 출력 강제 |
| 사실성 불안 | 근거 요구 없음 | 사실/해석/추정 분리 지시 |
| 답변 길이 과다 | 길이 제약 없음 | 최대 길이/항목 수 제한 |

## 디버깅 절차 (권장)
1. 동일 입력으로 재실행
2. 출력 형식 강제 추가
3. 근거 요구 문장 추가
4. 범위/금지사항 명확화
5. 결과를 테스트로 검증

## 서비스 전환이 나은 경우
- IDE 실시간 코드 자동완성 중심이면: Copilot 우선
- 대규모 리서치/링크 수집 중심이면: Gemini/Genspark 병행
- 고정밀 코드 패치 중심이면: Codex 계열 병행
