## Overview

OpenAI / GPT is the parent section for OpenAI's ChatGPT app, OpenAI API, and Codex surfaces. ChatGPT is the consumer and workspace app, OpenAI API is the developer platform surface, and Codex is the coding agent/product surface connected to ChatGPT plans, Codex credits, CLI workflows, and developer tooling.

- [Codex](codex/README.md): Codex CLI/app, changelog, rate card, ChatGPT integration, and OpenClaw OAuth usage facts.
