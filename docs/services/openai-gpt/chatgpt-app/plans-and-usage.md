## Billing Rules To Remember

| Rule | Detail | Official source | Last checked |
| --- | --- | --- | --- |
| ChatGPT app subscription and OpenAI API billing are separate. | Plus does not include API usage; API usage is billed independently. | [What is ChatGPT Plus?](https://help.openai.com/en/articles/6950777-chatgpt-plus) | 2026-04-26 |
| Go, Plus, and Pro do not support annual billing. | OpenAI states it does not support annual billing or paying multiple months in advance for those subscriptions. | [ChatGPT Pro tiers](https://help.openai.com/en/articles/9793128-what-is-chatgpt-pro) | 2026-04-26 |
| Business has a 2-user minimum. | The Business billing article states the subscription requires a minimum of 2 users. | [Manage billing on ChatGPT Business](https://help.openai.com/en/articles/8792536-what-is-chatgpt-business) | 2026-04-26 |
| Enterprise workspace access and API Platform access are separate. | Being in a ChatGPT Enterprise workspace does not automatically make the user a member of the API Platform organization. | [What is ChatGPT Enterprise?](https://help.openai.com/en/articles/8265053-what-is-chatgpt-enterprise) | 2026-04-26 |
