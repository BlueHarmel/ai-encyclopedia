# ChatGPT / Plans And Usage

Last verified: 2026-04-26

## What The Official Docs Say

This page explains the official ChatGPT plan and subscription facts that affect a paid ChatGPT user. OpenAI documents ChatGPT Plus and ChatGPT Pro as separate paid plans, and the Pro article separates Plus, Pro $100, and Pro $200 tiers. The facts below should be used for plan comparison, billing checks, and subscription management questions.

## How To Use This Page

- Use this page when the question is about monthly price, plan tier, usage allowance, cancellation, invoices, refunds, or annual billing.
- Do not use this page to infer API billing. OpenAI documents API usage as separate from ChatGPT Plus billing.
- Temporary Codex usage promotions must keep their official source and last-checked date because they can change.
