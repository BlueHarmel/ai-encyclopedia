# OpenAI / GPT / Codex / ChatGPT Integration

Last verified: 2026-04-26

## Summary

Codex 통합은 rate card, changelog, ChatGPT release notes가 섞인다. 사용량/credit과 기능 변경을 나눠 보는 표가 맞다.

## What The Sources Say

- The Codex rate card says Codex pricing was updated on April 2, 2026 to align with API token usage instead of per-message pricing. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card says the April 23, 2026 token-based pricing update applies to all existing ChatGPT Enterprise plans including Edu, Health, Gov, and ChatGPT for Teachers. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card says token-based pricing applies to new and existing ChatGPT Plus and Pro customers. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card says Codex usage is priced based on API token usage for input tokens, cached input tokens, and output tokens. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card lists GPT-5.5 at 125 credits per 1M input tokens, 12.50 credits per 1M cached input tokens, and 750 credits per 1M output tokens. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card says code review uses GPT-5.3-Codex. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card says GPT-5.3-Codex-Spark may be available in Codex as a research preview and credit rates are not final. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The Codex rate card says users can monitor token usage in Codex settings > Usage. ([Codex rate card](https://help.openai.com/en/articles/20001106))
- The March 26, 2026 ChatGPT release notes say Codex includes a curated plugins directory for packaged workflows built from apps and skills. ([ChatGPT release notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes))

- The April 9, 2026 release notes introduce a new $100 per month Pro plan and updated Codex usage across Plus and Pro. ([ChatGPT release notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes))
