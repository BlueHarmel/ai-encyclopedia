## Overview

Gemini는 Gemini app, Gemini API docs, API changelog, pricing page, Google blog 계열의 성격이 다르다. API 문서는 개발자 제품 기준이고, 앱과 블로그는 같은 사실로 바로 합치지 않는다.

## How This Section Is Organized
