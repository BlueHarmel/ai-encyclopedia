  • การเรียกเก็บเงินตามโทเค็นเอกสาร: ระบบจะเรียกเก็บเงินโทเค็นสำหรับรูปแบบ DOCUMENT (เช่น PDF) ในอัตราโทเค็นรูปภาพ ในการตอบกลับจาก API โทเค็นเหล่านี้จะปรากฏในรูปแบบ DOCUMENT ภายใน promptTokensDetails
  • การใช้งาน Google AI Studio ไม่มีค่าใช้จ่ายในภูมิภาคที่พร้อมให้บริการทั้งหมด ดูรายละเอียดได้ที่คำถามที่พบบ่อยเกี่ยวกับการเรียกเก็บเงิน
  • ราคาอาจแตกต่างจากราคาที่ระบุไว้ที่นี่และราคาที่เสนอใน แพลตฟอร์มเอเจนต์ Gemini Enterprise ดูราคาแพลตฟอร์ม Agent ของ Gemini Enterprise ได้ที่หน้าการกำหนดราคาแพลตฟอร์ม Agent ของ Gemini Enterprise
  • หากคุณใช้การดึงข้อมูลแบบไดนามิกเพื่อเพิ่มประสิทธิภาพค่าใช้จ่าย ระบบจะเรียกเก็บเงินเฉพาะคำขอที่มี URL การสนับสนุนการเชื่อมต่อแหล่งข้อมูลอย่างน้อย 1 รายการจากเว็บในคำตอบสำหรับการเชื่อมต่อแหล่งข้อมูลกับ Google Search ส่วนค่าใช้จ่ายสำหรับ Gemini จะมีผลเสมอ และโควต้าอัตราอาจมีการเปลี่ยนแปลง
