  • 동영상-이미지 생성 지원: 이제 텍스트 프롬프트와 함께 동영상 파일 (직접 업로드 또는 공개 YouTube URL을 통해)을 멀티모달 컨텍스트로 전달하여 고품질 썸네일, 영화 같은 영화 포스터 또는 요약 인포그래픽을 생성할 수 있습니다. 이 기능은 gemini-3.1-flash-image 모델에서만 지원됩니다. 자세한 내용은 동영상에서 이미지 생성 가이드를 참고하세요.

  • 지원 중단 공지: gemini-3.1-flash-image-preview 및 gemini-3-pro-image-preview 모델은 지원 중단되며 2026년 6월 25일에 종료됩니다.

  • 에이전트형 및 코딩 작업에서 지속적인 프런티어 성능을 제공하는 Google의 가장 지능적인 모델인 Gemini 3.5 Flash의 정식 버전 (GA)인 gemini-3.5-flash가 출시되었습니다.

  • Gemini API의 관리형 에이전트가 공개 프리뷰로 출시되었습니다. 이를 통해 개발자는 안전하고 격리된 Google 호스팅 Linux 샌드박스 환경에서 실행되는 자율적이고 상태 저장 에이전트를 빌드하고 배포할 수 있습니다. 자세한 내용은 에이전트 개요 페이지와 빠른 시작을 참고하세요.

  • 범용 Antigravity Agent 관리형 에이전트 antigravity-preview-05-2026가 공개 미리보기로 출시되었습니다. Antigravity 에이전트는 샌드박스 컨테이너 내에서 자율적으로 코드를 계획, 추론, 작성, 실행하고, 파일을 관리하고, 웹을 탐색할 수 있습니다. 코드 샘플과 사양은 Antigravity Agent 가이드를 참고하세요.

  • 지원 중단 공지: gemini-3.1-flash-lite-preview 모델은 2026년 5월 11일에 지원이 중단되며 2026년 5월 25일에 종료됩니다.

      • 정상적인 연결 해제 알림: 연결이 종료되기 직전을 나타내는 GoAway 서버 메시지를 수신하여 종료 전에 정상적으로 처리할 수 있습니다.

    • 구성 가능한 중단 처리: 사용자 입력이 모델의 대답을 중단해야 하는지 결정합니다.

    • 구성 가능한 미디어 해상도: 입력 미디어의 해상도를 선택하여 품질 또는 토큰 사용량을 최적화합니다.

    • 더 풍부한 출력 및 기능

    • 확장된 음성 및 언어 옵션: 오디오 출력을 위해 2개의 새로운 음성과 30개의 새로운 언어 중에서 선택할 수 있습니다. 이제 speechConfig 내에서 출력 언어를 구성할 수 있습니다.

    • 텍스트 스트리밍: 텍스트 응답이 생성될 때 점진적으로 수신하여 사용자에게 더 빠르게 표시할 수 있습니다.

  • aqa: 생성된 답변을 그라운딩하기 위해 텍스트 구절을 사용하여 질문에 답변하도록 학습된 새로운 특별 조정 모델입니다.

자세한 내용은 Gemini 모델을 참고하세요.

API 버전 업데이트:

  • v1: 안정적인 API 채널입니다.
  • v1beta: 베타 채널입니다. 이 채널에는 개발 중인 기능이 있을 수 있습니다.

자세한 내용은 API 버전 주제를 참고하세요.

API 업데이트:

  • GenerateContent는 채팅과 텍스트를 위한 단일 통합 엔드포인트입니다.
  • StreamGenerateContent 메서드를 통해 스트리밍을 사용할 수 있습니다.
