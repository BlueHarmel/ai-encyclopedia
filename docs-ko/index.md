# AI 백과사전

Last verified: 2026-04-26

이 백과사전은 AI 서비스를 추상적으로 소개하는 페이지가 아니라, 공식 출처 기반으로 실제 사용과 운영 판단에 필요한 사실을 정리하는 매뉴얼입니다.

## 읽는 순서

| 영역 | 언제 읽는가 | 상태 |
| --- | --- | --- |
| OpenAI / GPT | ChatGPT 앱, OpenAI API, Codex를 구분해서 확인할 때 | covered |
| Claude | Claude 앱, Claude Code, Anthropic API facts를 확인할 때 | indexed |
| Gemini | Gemini 앱과 Gemini API billing/API behavior를 분리해서 볼 때 | covered |
| GitHub Copilot | Copilot 플랜, IDE 기능, GitHub.com 기능, coding agent를 볼 때 | indexed |
| Genspark | 공식 문서가 부족한 영역과 확인 필요 영역을 볼 때 | needs-recheck |
| OpenClaw | 우리 PC의 OpenClaw 운영 기준, Telegram, cron, memory를 볼 때 | local-private-summary |
| People | Karpathy, Yann LeCun, Andrew Ng 같은 인물 signal을 볼 때 | signal |

## 원칙

- 공식 문서, 공식 릴리즈 노트, 공식 가격/플랜 문서를 fact source로 사용합니다.
- X, 뉴스, 커뮤니티, 교육 자료는 signal로 기록하고, 제품 사실로 승격하지 않습니다.
- 가격, 사용량, 모델 availability는 last checked 날짜를 붙입니다.
- 한국어 번역이 없는 페이지는 영어 원문을 표시합니다.
