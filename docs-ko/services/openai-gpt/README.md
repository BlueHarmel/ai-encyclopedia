# OpenAI / GPT

Last verified: 2026-04-26

OpenAI / GPT 섹션은 ChatGPT 앱, OpenAI API, Codex를 하나의 생태계 안에서 보되 서로 다른 표면으로 분리합니다.

## 하위 섹션

| 섹션 | 범위 | 읽는 이유 |
| --- | --- | --- |
| ChatGPT App | 소비자/워크스페이스용 ChatGPT 앱 | 플랜, 모델, 도구, 파일, 커넥터, 릴리즈 노트를 확인 |
| OpenAI API | 개발자 플랫폼과 API | ChatGPT 앱과 API billing/endpoint/tool behavior를 구분 |
| Codex | 코딩 에이전트와 개발 workflow | CLI, changelog, rate card, ChatGPT plan 연동, OpenClaw OAuth를 확인 |

## 경계 규칙

- ChatGPT 구독이 API 사용량을 의미하지는 않습니다.
- Codex는 ChatGPT 플랜/credit과 연결되지만, CLI와 개발자 workflow도 가진 별도 product surface입니다.
- 모델 이름이 같아도 앱 availability와 API availability는 따로 확인합니다.
