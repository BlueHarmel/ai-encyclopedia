Genspark

7. 기초 예제(따라하기)

실행 전 체크리스트

  • 출력 형식을 요약/근거 링크/확인 필요 3단으로 고정했는가?
  • 공식 Source 도메인 우선 규칙을 명시했는가?
  • 검증 불가 링크는 단정 문장에서 자동 제외하도록 설정했는가?

예제 1) 링크 포함 요약 만들기

요소내용
상황주간 리서치 노트를 빠르게 공유해야 한다.
입력주제 1개 + 공식/검증 링크 3~5개
출력요약 8줄 + 근거 링크 목록 + 확인 필요 항목
검토포인트단정 문장마다 링크가 연결되는지, 중복 링크가 제거됐는지 확인
실패패턴링크 없는 단정 문장을 포함해 검토 반려

예제 2) 비교표 초안 생성

요소내용
상황서비스 3개 비교 결정을 당일 회의에서 내려야 한다.
입력비교 대상 3개 + 기준(기능/제약/리스크) + 공식 링크
출력3열 비교표 + 항목별 근거 링크 + 확인 필요 행
검토포인트확인 불가 항목이 표 하단에 분리되는지 확인
실패패턴403 링크를 사실 근거로 사용해 잘못된 결론 도출

Source: Genspark 공식 사이트, genspark-ai/sparkle, genspark-ai/omaha

확인 불가/직접 확인 필요: 공식 사이트 세부 기능(검증 환경 403 가능), 플랜별 정책.
