Gemini

7. 기초 예제(따라하기)

실행 전 체크리스트

  • 출력 구조를 요약/근거 링크/확인 필요로 고정했는가?
  • 공식 Source 링크 없는 문장을 자동 분리하도록 지시했는가?
  • 배포 전에 사실/해석/추정 3분할 검토를 수행했는가?

예제 1) 문서 요약 + 확인 필요 분리

요소내용
상황공지/정책 문서를 비기술 독자에게 빠르게 공유해야 한다.
입력원문 문서 1개, 독자 수준(비기술), 길이 제한(핵심 5줄)
출력핵심 5줄 + 확인 필요 3개 + 근거 링크 목록
검토포인트링크 없는 문장이 확인 필요로 이동됐는지 확인
실패패턴요약 문장에 근거 링크가 누락되어 검토 반려

예제 2) 공지문 10줄 변환

요소내용
상황긴 브리프를 메신저 공지로 압축해야 한다.
입력원문 브리프, 길이 제한(10줄), 용어 단순화 규칙
출력공지문 10줄 + 링크 1개 + 직접 확인 필요 항목
검토포인트기술 용어 과다 여부, 핵심 행동 요청 문장 포함 여부
실패패턴길이 제한 미준수로 공지 가독성 저하

Source: Gemini API Docs, Gemini API Quickstart, Gemini Updates(verified)

확인 불가/직접 확인 필요: 계정/플랜별 고급 기능 제공 범위, 지역별 정책 차이.
