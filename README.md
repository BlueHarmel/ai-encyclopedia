개인 구독/운영 중인 AI 서비스들을 한곳에서 확인하기 위한 source-backed AI 백과사전입니다.

## 읽는 순서

- `docs/index.md`: 전체 지도와 운영 원칙
- `docs/services/*/README.md`: 서비스별 매뉴얼
- `docs/sources/source-registry.md`: 공식 문서, 릴리즈 노트, 뉴스룸, 교육자료
- `docs/sources/people-signal-registry.md`: AI 산업 주요 인물의 공개 발언 추적 대상
- `docs/workflows/source-to-publication.md`: 수집부터 게시까지의 자동화 흐름

## 원칙

- 공식 문서와 릴리즈 노트는 사실 원천으로 취급한다.
- 블로그, 뉴스, X 포스트, 커뮤니티 글은 맥락과 신호로 취급하며 공식 출처로 승격하지 않는다.
- 서비스 문서는 같은 템플릿을 강요하지 않고, 실제 사용자가 확인하려는 질문 중심으로 구성한다.
- 모르는 내용은 추정하지 않고 `needs-recheck`로 남긴다.
